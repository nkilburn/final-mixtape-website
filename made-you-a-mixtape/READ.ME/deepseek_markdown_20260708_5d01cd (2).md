# Made You A Mixtape – Podcast Website

A modern, multi‑page podcast site built with vanilla HTML, CSS, and JavaScript.  
Design inspired by Framer Launch, Sleepwave, and Song Exploder – featuring a floating pill navigation, animated gradient curtain hero, Song Exploder‑style audio player, studio channel strip contact form, and a dynamic J‑card tracklist that fills as you release episodes.

## Pages
- **Home** – Hero, audio player, episode detail, studio form
- **Episodes** – Dynamic J‑card (Season 1), episode archive grid with mini players
- **Blog** – Editorial card grid
- **Merch** – Product showcase
- **About** – Show & host info
- **Contact** – Studio channel strip form

## Quick Start
1. Clone the repository
2. Open `index.html` in any browser – no build tools required
3. To add episodes, edit the `season1Tracks` array in the `<script>` tag inside `index.html` (line ~3100)
4. Customize `feed.xml` with your actual podcast metadata and audio file URLs
5. Replace placeholder images/album art as needed
